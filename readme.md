## New Setup ##

Create Env

```
> mkdir myproject
> cd myproject
> py -3 -m venv .venv
```

Activate env

```
.venv\Scripts\activate
```

Install Flask

```
pip install Flask
```


## Restart App ## 

=> just choose the python interpreter in VSCode, set it up to use the venv that was setup above.
